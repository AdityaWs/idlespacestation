What is Idle Space Station?

idle space station is an idle game made by Aditya Wahyu Santoso using c++ and SDL. it's a game where your objective is to build the best space station ever. at the begining of the game you start with nothing but as you progress to the game you will eventualy unlock everything. it's a pretty fun game in my opinion and it's the 2nd game i made using c++
